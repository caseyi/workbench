# Workbench

Cross-thread task tracking. **TASKS.md is the canonical task list** — read/written live by the Cowork workbench-task-board artifact, weekly review runs Mondays 8am.

Sections: Inbox (untriaged, dropped by threads) / Active / Waiting On / Someday / Done.
Item format: `- [ ] **Title** - context #tag (from: thread) (touched: YYYY-MM-DD)` — Waiting On uses `(since: date)`.
Keep all context on one line; handoff files live in the local openelis-work/workbench/handoffs/ folder.
